### Step 1: Create a Special Repository
1. Go to [GitHub](https://github.com) and create a **new repository**
2. Name it **exactly** `PallaviJadar` (same as your GitHub username)
3. Make it **Public**
4. Check "Add a README file"
5. Click "Create repository"

### Step 2: Add the README Content
1. Copy the content from `README.md` in this folder
2. Go to your new `PallaviJadar` repository on GitHub
3. Click on the `README.md` file
4. Click the pencil icon (Edit this file)
5. Replace all content with the copied content
6. Scroll down and click "Commit changes"

### Step 3: Customize Your Profile
Update the following sections in your README:

#### Social Links (Bottom of README)
Replace these placeholder links with your actual links:
- LinkedIn: `https://linkedin.com/in/pallavi-jadar`
- Portfolio: `https://your-portfolio-url.com`
- Email: `your.email@example.com`

#### Featured Projects
Update the featured project section to showcase your best work:
```markdown

```

### Step 4: Verify Everything Works
1. Visit your GitHub profile: `https://github.com/PallaviJadar`
2. You should see your new README displayed prominently
3. Check that all images load correctly
4. Verify all links work

## üé® Customization Options

### Change Color Theme
The current theme uses a pink/radical color scheme (`#F75C7E`). To change colors:
- Find all instances of `theme=radical`
- Replace with: `theme=dark`, `theme=tokyonight`, `theme=dracula`, `theme=gruvbox`, etc.
- Find all instances of `F75C7E` (pink color)
- Replace with your preferred hex color (without the #)
